🌐 OhAPinger

Lightweight cross-platform network latency testing tool written in C++ (Clang-ready)

OSS project by bangkkuser

⬇ Download Latest Release

📦 Features

  • Ping any host or IP
  • Latency per request (ms)
  • Live mode (continuous ping stream)
  • Min / Max / Average stats
  • Packet loss calculation
  • Cross-platform (Windows / Linux / macOS / Android)

⚙️ Build (Clang)

clang++ main.cpp -O2 -std=c++17 -o ohaping

▶️ Run

./ohaping (Linux / macOS / Android)

📡 Live Mode

Type live instead of number of tests.

  • Continuous ping stream
  • Real-time latency output
  • Stop with CTRL + C
